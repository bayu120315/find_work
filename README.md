## 大学時代に作成したものの詳細について
【利用言語】HTML、CSS、Ruby
【開発環境】大学設置のPCにて開発。テキストエディタはEmacsを使用。
【作品の詳細】小説を選択肢によって枝分かれさせ、選択肢によって物語の結末が変わるようにしました。作成したRubyプログラムをCGIで動作させるように設計しました。
【チーム構成】メンバー7名(役割兼任あり)プログラム担当3名、詳細Webページ担当1名、小説担当2名、ポスター担当1名、パンフレット担当3名
【担当領域】プログラミング、詳細Webページ
【工夫したところ】全てのページ数と、現在のページ数を記載し、クリアまであと何ページなのか分かりやすくしました。ゲームオーバーになった際に、最初のページに戻るか、前のページに戻るかの選択肢を用意し、より楽しめるようにしました。
